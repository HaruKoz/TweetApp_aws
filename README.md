# TweetApp
フレームワークを使用せず、プレーンなPHPで作成したTwitter風Webアプリケーションです。

# 使用技術
PHP(7.2.34)

HTML

CSS

MariaDB(10.2.38)

JavaScript(jQuery)

AWS(IAM, VPC, EC2, Route53, Certificate Manager)

CircleCI

# 機能一覧
- サインアップ
- ログイン、ログアウト
- ツイート（投稿、削除）
- いいね
- おすすめユーザー表示
- アカウント設定編集
